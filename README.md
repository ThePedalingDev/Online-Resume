# Online Resume (Vite + React + Tailwind)

A personal portfolio / resume site built with Vite, React, TailwindCSS, DaisyUI, and Framer Motion.

## Prerequisites
- Node.js 18+
- npm 9+

## Setup
```bash
npm install
```

## Development
```bash
npm run dev
```
Vite will start a dev server (default: http://localhost:5173).

## Build
```bash
npm run build
```
Outputs a production build to `dist/`.

## Preview (serve build locally)
```bash
npm run preview
```

## Tech Stack
- React + Vite
- TailwindCSS + DaisyUI
- Framer Motion
